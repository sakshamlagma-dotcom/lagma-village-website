# Visitor Tracking Setup

Visitor tracking code is already added to the website. To activate it:

1. Open Google Analytics and create a property for this website.
2. Create a Web data stream for `https://sakshamlagma-dotcom.github.io/lagma-village-website/`.
3. Copy the Measurement ID. It looks like `G-ABC123XYZ`.
4. Replace `G-XXXXXXXXXX` in every HTML page with your real Measurement ID.
5. Commit and push the files to GitHub.

After publishing, Google Analytics will show visitors, active users, popular pages, devices, locations, and traffic sources.
